indexOf/lastIndexOf

arr = ["사과", "바나나", "복숭아", "사과", "딸기" ,"포도","사과", "복숭아"]
indexOf(item,start) : item의 위치를 검색.
start : 시작위치, 생략시 0번째부터 시작
lastIndexOf(item) : item의 위치를 뒤에서 부터 검색

push/pop

push : 배열의 끝에 추가
pop : 배열의 끝에서 제거

reverse/slice

reverse(): 배열의 순서를뒤집기
slice(start, end) : start end 사이의 값을 추출, 새로운 배열 만들기

splice

splice() : 배열 수정
- splice(idx, 0, 새로운데이타) : 기존데이타 삭제하지 않고 새로운 데이타를 삽입
- splice(idx, 삭제갯수, 새로운데이타) : 기존데이타 삭제후 새로운 데이타를 삽입
- splice(idx, 삭제갯수) : 기존데이타를 삭제
- splice(idx, 1) : 삭제할 데이터를 입력 받는다.

sort

arr = [ 211, 11, 100, 34, 314, 500, 58 ];
- 자릿수 별로 정렬 : 100, 11, 211, 314, 34, 500, 58
- 크기별로 오름차순 정렬 : 11, 34, 58, 100, 211, 314, 500
sort(function(a,b)){
return a-b;
}
- a-b의 결과 값이 + 일때 자리를 바꾼다.
- b-a로 하면 자리가 바뀐다.

concat()/join

concat() : 여러개의 배열을 하나로 합치기
join(구분자) : 하나의 배열에서 각 요소를 구분할 구분자로 구분한다.
