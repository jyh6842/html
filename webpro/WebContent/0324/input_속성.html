속성
required : 반드시 입력해야한다는 필수 항목 
readonly : 입력불가
placeholder : 예시를 표시하는 것 
autofocus :  새로고침하면 자동으로 커서가 되어있는 것
pattern : 데이터의 입력 규칙
        [0-9]0~9 사이의 숫자
        [a-z]a~z 사이의 문자
        [ㄱ-ㅎ]
        [가-힣]
        {3}  3자리
        {2,5} 2~5자리
title : (input태그 안에서는 제목이 아닌)말풍선      
button은 원래 기능이 없지만 기능을 주려면 안에  onclick=""을 넣어주면 기능이 살아난다.
