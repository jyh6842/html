		<labal>label</labal>적용
		label은 inline 요소 이다.
		
		인라인 속성을 블럭속성으로 변경한다. 왜? 변경 : inline은 크기 조절이 불가능하다. 크기를 조절하기 위해서는 block으로 변경해야 한다.
		display 스타일을 설정
		inline	: 줄이 바뀌지 않는다.
		block	: 자동으로 줄바꿈, 크기조절 할 수 있다.
		inline-block	: 줄이 바뀌지 않으면서 크기 조절을 할 수 있다.
		none	: 화면에 표시하지 않는다. ;
